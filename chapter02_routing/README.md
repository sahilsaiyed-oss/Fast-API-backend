


# 📘 THEORY 

## 🔹 Definition  
Routing is the process of mapping URLs to specific functions in FastAPI.

## 🔹 Why Use Routing?  
- Organize API  
- Handle multiple requests  
- Create dynamic endpoints  

## 🔹 Types of Routes  
- Static → `/about`  
- Dynamic → `/user/{username}`  

## 🔹 What is Path Parameter?  
A value passed inside URL.
