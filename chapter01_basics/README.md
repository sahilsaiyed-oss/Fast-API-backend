# Chapter 01 - FastAPI Basics

## 🔹 1. Definition
FastAPI is a high-performance Python framework used to build APIs.

## 🔹 2. Why Use It?
- Fast (async support)
- Automatic API docs
- Easy to build APIs

## 🔹 3. Key Concepts
- API: Communication between client & server
- Endpoint: URL where request is sent
- GET: Used to fetch data

## 🔹 4. Code Understanding
- FastAPI() → creates app
- @app.get("/") → defines route
- Function → returns response

## 🔹 5. Output Example
GET / → {"message": "Welcome"}

## 🔹 6. Interview Points
- FastAPI is based on Starlette & Pydantic
- Supports async programming
- Automatically generates Swagger UI (/docs)
